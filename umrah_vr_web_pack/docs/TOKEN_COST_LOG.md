# Token and AI Cost Strategy

## Principle
Do not use AI for normal wizard viewing. Use AI only for explicit user actions.

## Free / Low Cost Features
- Static translations.
- Pre-recorded audio.
- Browser text-to-speech.
- Local recording.
- Local progress.
- Static 360 scenes.

## Paid AI Features
- Translate my personal dua.
- Simplify this step for children.
- Ask an educational question.
- Generate a personalized travel checklist.

## Cost Controls
- Cache AI responses.
- Require user click before AI call.
- Limit answer length.
- Use smaller models for translation/simplification.
- Disable AI by default for agency white-label version.
