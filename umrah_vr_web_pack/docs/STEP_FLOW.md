# 15-Step Umrah Wizard Flow

1. Welcome and journey setup
2. Prepare for Ihram
3. Make Niyyah at Miqat
4. Recite Talbiyah
5. Enter Masjid al-Haram
6. Start Tawaf
7. Complete 7 rounds of Tawaf
8. Pray 2 Rak'ahs
9. Drink Zamzam
10. Sa'i between Safa and Marwa
11. Trim or shave hair
12. Umrah complete / exit Ihram
13. Optional recommended deeds
14. Final dua and departure from Makkah
15. Visit Madinah / journey reflection

Each step should include:
- title
- short subtitle
- hero image or 360 scene
- explanation
- checklist
- audio script
- personal recording button
- personal dua text field
- language selector
- next/back buttons
