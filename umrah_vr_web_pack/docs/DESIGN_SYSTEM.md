# Design System

## Visual Style
- White/cream background.
- Deep green primary color.
- Gold accent for headings and spiritual emphasis.
- Rounded cards.
- Large readable typography.
- Mobile-first layout.

## Main Components
- StepProgressBar
- LanguageSwitcher
- AudioPlayerButton
- PersonalRecordingCard
- PersonalDuaEditor
- StepHeroMedia
- ChecklistCard
- Scene360Viewer
- HotspotInfoCard
- AdminReviewBadge

## Accessibility
- Large text mode.
- Audio speed controls.
- High contrast option.
- RTL support for Arabic and Urdu.
- Captions/subtitles for audio.
