# Umrah Virtual Trip Web Platform

A low-cost, web-first virtual Umrah training and guidance platform for integration into an existing website. It uses 360 images/video and lightweight 3D only where useful, with multilingual step screens, listening mode, pilgrim personal recordings, personal duas, and an optional MCP server for content/admin operations.

Recommended product name: Umrah Journey Guide

Core approach: do not build expensive full VR first. Start with a browser-based 360 tour + wizard, then add WebXR/VR headset support later.
