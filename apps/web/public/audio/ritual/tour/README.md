# Virtual-tour narration audio (optional)

Per scene + language, named <scene-id>.<lang>.mp3 — e.g. kaaba.en.mp3, kaaba.ar.mp3, safa-marwa.ur.mp3.
Languages: en, ar, ur, tr, de. If a file is absent, the Listen button reads the on-screen description
aloud via the browser (text-to-speech) in the selected language. Keep clips short and web-optimized.
