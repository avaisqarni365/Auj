# Virtual-tour walkthrough videos

Drop a short walkthrough video per tour scene here, named by scene id (web-optimized MP4, H.264/AAC,
≤ ~10 MB, landscape). They appear side-by-side with the panorama on /guide/tour automatically; until a
file exists the panel shows a calm "coming soon" placeholder.

Filenames: miqat.mp4 · haram-entrance.mp4 · kaaba.mp4 · maqam-ibrahim.mp4 · zamzam.mp4 ·
safa-marwa.mp4 · completion.mp4 · madinah.mp4
