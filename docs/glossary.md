# Glossary — domain terms

- **Umrah / Hajj** — the minor (any-time) and major (fixed-date) pilgrimages to Makkah.
- **Nusuk** — Saudi Arabia's official pilgrimage platform (umrah.nusuk.sa).
- **Nusuk Masar / Masar Nusuk** — the B2B / external-agent platform behind Nusuk.
- **Maqam GDS** — the Ministry's distribution system for hotels, transport, ground services; gated.
- **BRN** — Booking Reference Number generated per booking; required for visa processing.
- **Naqaba** — the licensed transport network (inter-city + transfers) reached via the connector.
- **External agent (EA)** — a foreign agent authorized (with an agent code) to use Masar Nusuk,
  contracted to exactly one Saudi-licensed Umrah company.
- **MoRA** — Pakistan's Ministry of Religious Affairs & Interfaith Harmony; licenses PK operators.
- **VVTAT** — Lithuania's State Consumer Rights Protection Authority; issues the tour-operator cert.
- **MB** — Mažoji bendrija, a Lithuanian small company; sufficient to hold the tour-operator cert.
- **PTD** — EU Package Travel Directive (insolvency protection, refund timelines).
- **e-visa** — Saudi online tourist visa (allows Umrah); eligibility follows passport/residence.
- **mahram** — a required adult guardian linkage for some pilgrims; modeled on Pilgrim.
- **ziyarah** — guided visits to religious sites, sold as an add-on.
